# MapsMap Prototype - Frontend

### Requirements

- Node v12 LTS

### Installation

```
git clone https://github.com/Lemii/mapsmap
cd ./mapsmap/frontend
npm i
```

### Usage

#### Run in development mode

```
npm start
```

#### Run in production mode

```
npm i -g serve
npm run build
cd build
serve -s
```
